Vocabulary Size:

🌱 Beginner (HSK 1-2): 80 words — Numbers, family, food, colors, common verbs, time, adjectives, places

🌿 Intermediate (HSK 3-4): 65 words — Social, work, travel, emotions, abstract concepts, daily advanced vocabulary

🌳 Advanced (HSK 5-6): 55 words — Idioms and allusions, literature and philosophy, formal language, advanced vocabulary

Two Modes:

📖 Review Mode — Default only displays Chinese characters; clicking the card reveals pinyin and definition. Forgot/Pass buttons control review frequency.

🔍 Beginner Mode — Click to reveal words step-by-step (Chinese character → pinyin → definition). Hard/Easy difficulty levels are marked; Hard words are automatically added back to the queue for reinforcement.

Features: Dark theme, mobile-friendly
Real-time statistics (to be learned/reviewed/completed) + progress bar
Forgotten/difficult words are automatically added to the list in levels 3-6 Zhang reappeared. There are pass rate statistics after completing the course. You can use it by simply opening the HTML file in a browser; it's pure front-end development with zero dependencies.
